HTML5图像
支持的图像格式：PNG、GIF、JPG、SVG
使用标签来插入图像
示例：
标签 描述
img 定义图像
map 定义图像映射
area 定义图像映射中的区域
figure 定义图像的组合
figcaption 定义图像组合的标题
标签的常用属性
属性 描述
alt 图像的替代文本
crossorigin 规定是否从其他域请求图像
height 图像高度
ismap 规定图像是否为服务器端图像映射
longdesc 规定图像的详细描述
src 图像的 URL
usemap 规定图像的客户端图像映射
width 图像宽度
HTML5视频
支持的视频格式：版权原因，只支持MP4、WebM、Ogv
使用

标签来插入表格
标签 描述
table 定义表格
caption 定义表格标题
colgroup 定义表格列组
col 定义表格列
thead 定义表格头部
tbody 定义表格主体
tfoot 定义表格尾部
tr 定义表格行
th 定义表格标题单元格
td 定义表格单元格
colspan 定义单元格横跨的列数
rowspan 定义单元格横跨的行数
标签的常用属性 属性 描述 align 规定表格的对齐方式 bgcolor 规定表格的背景颜色 border 规定表格的边框宽度 cellpadding 规定单元格内边距 cellspacing 规定单元格之间的间距 frame 规定表格的边框 rules 规定表格的边框 summary 规定表格的摘要 width 规定表格的宽度 HTML5表单 使用标签来插入表单 表单允许用户在浏览器中输入数据，然后将这些数据发送到服务器，服务器处理后返回响应。 表单控件：表单中的每个控件都是一个表单域。 标签 描述 form 定义表单 input 定义输入字段 textarea 定义多行输入字段 button 定义按钮 output 定义输出字段 label 定义输入字段的标签 fieldset 定义输入字段的组 legend 定义输入字段组的标题 keygen 定义用于表单的密钥对生成器字段 datalist 定义输入字段的选项列表 select 定义下拉列表 optgroup 定义下拉列表中的选项组 option 定义下拉列表中的选项 标签的常用属性 属性 值 描述 accept-charset 字符编码 规定在表单提交时使用的字符编码 action URL 规定当提交表单时向何处发送表单数据，不可为空 autocomplete on/off 规定是否启用自动完成功能 enctype application/x-www-form-urlencoded multipart/form-data text/plain 规定在发送到服务器之前,应该如何对表单数据进行编码 method get/post 规定用于发送表单数据的 HTTP 方法 name name 规定表单的名称 novalidate novalidate 规定是否在提交表单时不对其进行验证 target _blank _self get和post的区别 get: 数据在URL中 数据大小有限制，需要传递的数据量较小 数据不安全,会被浏览器缓存并保存历史记录 post: 数据在请求体中 数据大小无限制 数据安全，不会被浏览器缓存，也不会被保存历史记录。 表单域：表单中的每个控件都是一个表单域。 表单域 描述 文本域 允许用户输入文本 密码域 允许用户输入密码 单选按钮 允许用户从一组选项中选择一个选项 复选框 允许用户从一组选项中选择多个选项 下拉列表 允许用户从一组选项中选择一个选项 提交按钮 向服务器提交表单数据 重置按钮 重置表单中的所有控件 文件域 允许用户上传文件 隐藏域 存储页面信息 标签的常用属性 属性 值 描述 accept 文件类型 规定文件的类型 alt 文本 规定在图像无法显示时的替代文本 autocomplete on/off 规定是否启用自动完成功能 autofocus autofocus 规定在页面加载时，该元素应该自动获得焦点 checked checked 规定该元素应该被预先选中 disabled disabled 规定该元素应该被禁用 form form_id 规定该元素所属的表单 formaction URL 规定当提交表单时向何处发送表单数据 formenctype application/x-www-form-urlencoded multipart/form-data text/plain 规定在发送到服务器之前,应该如何对表单数据进行编码 formmethod get/post 规定用于发送表单数据的 HTTP 方法 max number 规定输入字段允许的最大值 maxlength number 规定输入字段允许的最大字符数 min number 规定输入字段允许的最小值 multiple multiple 规定输入字段允许多个值 name name 规定输入字段的名称 标签的常用属性 属性 描述 button 定义按钮 checkbox 定义复选框 color 定义颜色选择器 date 定义日期选择器 datetime-local 定义日期和时间选择器 email 定义电子邮件地址输入域 file 定义文件域 hidden 定义隐藏域 image 定义图像域 month 定义月份和年份选择器 number 定义数字输入域 password 定义密码域，其中字符会被掩码(显示为星号或者圆点) radio 定义单选按钮 range 定义滑动条 reset 定义重置按钮 search 定义搜索域 submit 定义提交按钮 tel 定义电话号码输入域 text 定义单行文本域,默认宽度为20个字符 time 定义时间选择器 url 定义 URL 地址输入域 week 定义周和年选择器
