命令行帮助

  • 任意命令模式下,输入”?”：获取该命令模式下所有命令及其简单描述
  • 在命令关键字后输入以空格分隔的”?”
  • if此位置是关键字是参数,会输出该参数类型、范围等描述
  • if此位置是关键字,则列出关键字集合及其简单描述

常用配置技巧

命令简写

支持缩写命令,即在输入命令和关键字时,只要输入的命令所包含字符长到足以与其他命令区别就足够了。（比如enable可写成en,interface ethernet 0/2 命令缩写为int e 0/2）

使用命令no选项

使用no选项,禁止某个特性或功能,或者执行与命令本身相反
tab键补全

命令查询

输入命令的部分字符串后键入”?”,显示与之匹配的所有命令

历史命令

交换机常用配置命令

交换机命名hostname

Switch(config)#hostname S1

show命令集

  • 显示IOS版本信息：Switch#show version
  • 显示所有vlan摘要信息：Switch#show vlan
  • 显示接口的IP信息：Switch#show ip interface brief
  • 显示正在运行的配置文件：Switch#show running-config
  • 显示MAC地址表：Switch#show mac-address-table
  • 显示交换机端口的信息：Switch#show interface ethernet 0/0/1

交换机与路由器密码配置

此处以交换机为例，路由器同理

密码用于控制用户进入不同的模式

  • 使用明文密码：Switch(config)#enable password pw123456
  • 使用加密密码：Switch(config)#enable secret pw123456
